Introduction to R

Welcome to Introduction to R, a six-week, in-person course held every Wednesday, tailored for researchers and students. This course covers the essentials of R programming, including data manipulation, statistical analysis, and data visualization, providing practical skills to support your research and academic projects.

About Introduction to R

Data analysis is an essential skill to enabling students and researchers to answer key questions in their studies through testing their hypothesis.

Supported by CEMA - University of Nairobi, the course covers key concepts in R programming, including data manipulation, statistical operations, and graphical representation. Participants will gain practical knowledge in working with data structures, importing and cleaning data, applying transformation techniques, and creating clear and effective visualization.

This course is ideal for those looking to apply R in academic research, professional projects, and data-driven decision-making.

Timetable

    Afternoon session: 2pm - 4pm


Week 1

    Chapter 1: Introduction to the R environment

  • This chapter introduces the R programming environment, covering the R console, RStudio interface, and basic commands.

    • Chapter 2: Getting started

  • This chapter focuses on creating and working with different R objects such as vectors, lists, matrices, data frames, and factors. It also covers data types like characters, booleans, numerics, handling missing data, and working with dates

Week 2

    Chapter 3: Simple R functions

  • Learners will explore basic R functions for arithmetic operations, statistical summaries, and mathematical transformations. This chapter covers commonly used functions like sum(), mean(), log(), and trigonometric functions

Week 3

    Chapter 4: The data.frame object

  • This chapter explains the structure and organization of data frames, Râ€™s most common data structure. It covers how to create, manipulate, and subset data frames, along with basic operations similar to working with Excel tables

    • ,

    Chapter 5: Data Importation

  • Participants will learn how to import data from various sources, including Excel and CSV files. The chapter introduces essential packages like readr, readxl, and the tidyverse, along with methods for installing and loading packages in R

Week 4

    Chapter 6: Data Manipulation

  • This chapter introduces data manipulation techniques using dplyr. Learners will practice operations like select(), filter(), mutate(), and arrange() to clean and transform datasets efficiently using the pipe operator (%>%, |>)

Week 5

    Chapter 7: Advanced Manipulation Functions

  • Building on previous chapters, this section covers advanced functions for reshaping data with pivot_longer() and pivot_wider(), merging datasets using join functions, and performing grouped summaries with group_by() and summarise()

Week 6

    Chapter 8: The Grammar of Graphics

  • This chapter introduces data visualization using ggplot2. Participants will learn to create various plots, including bar charts, line graphs, scatter plots, histograms, and box plots, along with techniques for faceting and adding trend lines
